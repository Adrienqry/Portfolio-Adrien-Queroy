# Portfolio Adrien Queroy

Portfolio statique en HTML, CSS et JavaScript, pense pour un deploiement simple sur Netlify.

## Lancer en local

Avec Python :

```bash
python -m http.server 8000
```

Puis ouvrir `http://localhost:8000`.

## Deployer sur Netlify

1. Creer un depot GitHub avec ces fichiers.
2. Connecter le depot a Netlify.
3. Laisser le dossier de publication sur `.`.
4. Renommer le site dans Netlify si tu veux une URL plus simple, par exemple `portfolio-adrien-queroy`.
5. Si un ancien asset reste affiche, lancer `Clear cache and deploy site`.
6. Deployer.

## Personnalisation rapide

- Modifier les textes dans `index.html`
- Ajuster l'identite visuelle dans `styles.css`
- Ajouter tes liens projets et ton GitHub dans la section contact
